# RFC directory

Design docs for M/L tier changes per `~/.claude/skills/dev-workflow/SKILL.md`.

Format: `NNN-title.md`, sequential.

## Template

```
## Проблема
What's broken / missing. 1–2 sentences.

## Варианты
- A: ...
- B: ...

## Выбрано + почему
Option X, because ...

## Риски
- ...

## План проверки
How we'll know it works (tests, manual scenario, metric).
```

For L-tier add: file architecture, contracts, edge cases, rollback,
out-of-scope.
